This is client-register app

There is 3 pages:
 - List of clients (/clients)
 - Registration form (/registration)
 - About me (/about)

You can easily navigate by the header on the site.
On the /clients page user can see the list of users or click on the register-btn to navigate to /registration page.
On the /registration page user can register himself to the page
On the /about page there is a text about author of this code.


### Installation

Install the dependencies and devDependencies and start the server.

```sh
$ npm i
$ npm run dev
```

For production environments...

```sh
$ npm run build
$ npm run start
```
